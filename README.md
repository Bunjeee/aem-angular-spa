# AEM angular application template
Scaffolding for angular project deployable into AEM environment

## Prerequisities
Node JS 6.9.2+

## Project initialization
<code>npm install</code>

## Run command
* <code>gulp build</code> - builds entire project and saves resources into <code>dist</code> folder.
* <code>gulp serve</code> - runs SPA application
* <code>gulp deploy</code> - deploys application into AEM environment
